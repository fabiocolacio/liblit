# liblit

liblit is a library for parsing and creating digital book formats.

Filetype Support:

Format | Parsing | Writing
-------|---------|--------
Epub   | [X]     | [ ]
Mobi   | [ ]     | [ ]
Cbz    | [X]     | [ ]
Cbt    | [ ]     | [ ]
Cbr    | [ ]     | [ ]

Example usage can be found in ``/examples``
